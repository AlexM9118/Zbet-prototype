# Zbet-prototype

Acest repo este un prototip separat pentru directia noua `match analyzer`.

Scop:
- sa experimentam un flow nou
- fara sa atingem produsul live actual
- fara call-uri noi in OddsPapi

Date folosite:
- snapshot local din proiectul principal:
  - `data/ui/matches.json`
  - `data/ui/history_stats.json`

Concept:
- utilizatorul alege competitia
- alege meciul
- poate analiza si toate meciurile din liga
- vede 1-2 recomandari
- poate extinde:
  - toate pietele
  - forma si comparatia
